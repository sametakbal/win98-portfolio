We're excited to announce the first release of Windows 98 Portfolio - a nostalgic, fully functional personal portfolio website that brings back the iconic Windows 98 aesthetic!

### ‚ú® Features

#### Core Experience

- **Authentic Windows 98 UI** - Pixel-perfect recreation of the classic Windows 98 interface
  - Classic window styling with title bars, borders, and buttons
  - Authentic taskbar with Start button and system tray
  - Working Start menu with submenus
  - Desktop icons with hover effects

#### Interactive Elements

- **Draggable Windows** - All windows can be freely moved around the desktop by dragging the title bar
- **Multiple Window Support** - Open and manage multiple windows simultaneously
- **Window Management** - Minimize, maximize, and close buttons (close fully functional)

#### Applications & Features

- **About Window** - Personal information showcase
  - Profile details (name, title, experience, specialization)
  - Social media integration (GitHub, LinkedIn, YouTube, Medium, Twitter)
  - Professional bio section
  
- **Projects Window** - Portfolio showcase with Windows Explorer styling
  - Project cards with descriptions
  - Technology tags
  - Project years
  - Custom toolbar (File, Edit, View, Help)

- **Articles & Videos Window** - Content showcase
  - Support for Medium articles and YouTube videos
  - Platform indicators with emojis
  - Read time estimates
  - Direct links to content
  - Publication dates

- **Minesweeper Game** - Fully playable classic game
  - 9x9 grid with 10 mines
  - Left-click to reveal cells
  - Right-click to flag/unflag
  - Automatic cascade for empty cells
  - Mine counter
  - Timer display
  - Reset button with emoji feedback (üòµ game over, üòé win, üôÇ playing)
  - Win/lose detection

- **Internet Explorer Window** - Vintage web browsing experience
  - Classic IE interface with menu bar (File, Edit, View, Favorites, Tools, Help)
  - Address bar with URL display and Go button
  - Embedded iframe for web content
  - Pre-loaded with 1999 Google from Web Archive
  - Authentic Windows 98 IE styling

#### Internationalization

- **Multilingual Support** - Seamless language switching
  - English translation
  - Turkish translation
  - Easy to extend with additional languages
  - Language selector in Start menu
  - Persistent language preference

  - Window title display with emojis
  - System tray (volume icon, clock)
  - Real-time clock (updates every minute)
